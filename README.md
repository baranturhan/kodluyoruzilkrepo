## Installation

Öncelikle projeyi clonelayın.
'git clone https://github.com/baranturhan/kodluyoruzilkrepo.git'

## Usage

Projeyi cloneladıktan sonra Visual Studio Code programında açınız.

Mac için:
'''
    cd kodluyoruzilkrepo
    open README.md
'''
